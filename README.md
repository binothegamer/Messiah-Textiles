# Messiah Textiles

A modern landing page for Messiah Textiles, featuring premium quality fabrics and textiles.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Navigation Bar**: Sticky navbar with logo and menu items
- **Image Gallery**: Showcase of premium textile products
- **Interactive Tabs**: Product category information with tabbed interface
- **Accordion FAQ**: Expandable frequently asked questions section
- **Contact Footer**: Complete contact information and business hours

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies Used

- React 18
- Vite
- CSS3 (Custom styling)
